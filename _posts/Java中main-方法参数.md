Java中的main()方法

```java
public class HelloWorld {
    public static void main(String[] args) {
	}
}
```



在命令行下执行 Java 的 Class 时，此参数接收一个参数列表，形式为

```java
$ java ClassName arg1 arg2 arg3 ...
```
